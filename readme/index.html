    8. 环境优化配置

    8.1 开发环境优化

    HMR(模块热替换) 在未使用HMR的时候，webpack serve在自动更新的时候，会从新加载所有文件；

    1. 开启HMR后，样式的HMR功能，需要在开发环境使用style-loader
    2. HTML的HMR功能，默认也没有HMR功能，需要在entry中引入html文件

    在webpack.config.js中配置

        devServer: {
        hot: true,
    },

    在入口的js文件中增加一个判断

    if(module.hot){
    // 只对print.js进行热更新

    })

    10 axios与promise

    10.1 promise的使用

    主要用于异步计算，可以将异步操作队列化，按照期望的顺序执行；解决回调地狱

    // 此处传两个函数名
new Promise((resolve,reject)=>{
    console.log("这是第一层")
    // resolve和reject只能生效一个
    // 有优先级，从上到下
    // resolve表示true，会执行.then(par1,par2)的第一个参数所代表的箭头函数
    // reject表示false，会执行.then(par1,par2)的第二个参数所代表的箭头函数
    reject("这是一个错误")
    resolve("这是第二2层")
}).then(res=>{
    console.log(res)
    /*
        这里的res，是
        new Promise((resolve,reject)=>{})
        中的resolve函数的值
    */

    /*
        如果还想继续执行，可以return一个Promise
        return new Promise((resolve,reject)=>{

        })
    */
},err=>{
    console.log(err)
}
)

    .then继续执行的三种写法

    new Promise((resolve,reject)=>{
    console.log("这是第一层")
    resolve("这是第二2层")
}).then(res=>{
    console.log(res)

    // 方式一：
    // return new Promise((resolve,reject)=>{
    //     resolve("这是第三3层")
    // })

    // 方式二：
    // return Promise.resolve("这是第三层")

    // 方式三：

    11.7.3 计算属性computed

    有缓存作用

    <template>
	<div>
		<h2>{{ name }}-{{ slogen }}</h2>
		<h2>{{ name + '-' + slogen }}</h2>
		<!-- 此处2被调用了几次就打印几次 -->
		<h2>{{ getTitle() }}</h2>
		<h2>{{ getTitle() }}</h2>
		<h2>{{ getTitle() }}</h2>
		<!-- 此处不是调用方法，用的是属性，计算属性有缓存作用,此处1只打印了1次 -->
		<h2>{{ title }}</h2>
		<h2>{{ title }}</h2>
		<h2>{{ title }}</h2>
		<br />
		<small>￥{{ totalPrice }}</small>
	</div>
</template>

<script>
export default {
	name: 'App',
	data() {
		return {
			name: '学习',
			slogen: '成就自己',
			books: [
				{ id: 1, name: '《操作系统原理》', price: 188 },
				{ id: 2, name: '《操作系统真相还原》', price: 88 },
				{ id: 3, name: '《c语言从入门到入土》', price: 66 },
				{ id: 4, name: '《k8s》', price: 99 },
			],
		}
	},
	computed: {
		totalPrice: {
			get() {
				// reduce((s, n)中的s表示箭头函数右边的返回值， 0表示第一次s为0
				// 循环这个books，依次取出元素，并赋值给n；
				// 第一次表示 reduce(0,{ id: 1, name: '《操作系统原理》', price: 188 }) =>0+188
				// 第二次表示 reduce(188,{ id: 2, name: '《操作系统真相还原》', price: 88 }) => 188+88
				// 第三次表示 reduce(276,{ id: 3, name: '《c语言从入门到入土》', price: 66 }) => 276+66
				// 第四次表示 reduce(342,{ id: 4, name: '《k8s》', price: 99 }) => 342+99
				return this.books.reduce((s, n) => s + n.price, 0)
			},
		},
		title: {
			get() {
				console.log('1111111')
				return this.name + '-' + this.slogen
			},
			// set基本不用
			set(values) {
				let arr = values.split('-')
				this.name = arr[0]
				this.slogen = arr[1]
			},
		},
	},
	methods: {
		getTitle() {
			console.log('222222')
			// 此处调用的是computed中的set
			this.title = '学习 - slogen'
			return this.name + '-' + this.slogen
		},
	},
}
</script>
